Configuración de la Jetson Nano

Vamos a configurar la Jetson Nano, primero tenemos que insertar la micro SD para poder instalar ubuntu en su m.2 ya que no tiene ningún sistema operativo dentro.

una vez lo introducimos, conectamos la Jetson Nano a la corriente para poder trabajar con el y instalarle el sistema operativo. Una vez lo encendamos nos debe aparecer algo así

y no nos tiene que salir ningún error, ya que eso significaría que el sistema operativo lo hemos instalado mal en la microsd, una vez que termina nos aparece el sistema operativo pero pidiendonos la confuguración inicial. Donde lo primero que nos pide es leer y aceptar las licencias.

Le damos a aceptar y continuamos con la configuración, indicamos el idioma, 

la disposicion del  teclado, en nuestro caso en español de españa.

después nos pide conectarnos a internet, la cual elegimos el wifi de la clase 

Después del wifi, nos indica que tenemos que poner una localización, esto sirve sobretodo para el horario, para que se configure automaticamente y no tengamos que cambiarla cada vez

Despues nos indica de poner el usuario y la contraseña, ponemos de usuario aulaateca26

despues de configurar el usuario y la contraseña, nos indica que cuanto queremos usar del disco m.2 en nuestro caso usaremos todo, ya que no vamos a hacer un dualboot ni nada por el estilo.

Ahí vemos como esta instalando todo, y ahora a esperar a que se instale.

Ahí tenemos la jetson nano funcionando.

Ahí indicamos skip, y continuamos abrimos chromium y vemos que no funciona

Para lograr que funcione lo que tenemos que hacer es actualizar todos los paquetes y demás, esto va a ser fácil ya que vienen en contenedores lo que nos facilita a la hora de actualizarlos.

una vez actualizamos todo abrimos chromium y si nos funciona.

después conectamos el ratón y el teclado inalambrico de logitech.

Una vez terminado de conectar ya tenemos la configuración básica terminada.

Ahora vamos a entrar a la página principal de la Jetson Nano, para poder configurar el nvme m.2 

y vamos a Jetson Setup guide -> SSD + Docker

 Después nos vamos a la terminal, donde escribimos lspci que sirve para mostrar información detallada sobre todos los dispositivos conectados al bus PCI (Peripheral Component Interconnect) y PCI Express (PCIe).

Después escribimos lsblk que sirve para listar información sobre todos los dispositivos de bloques de almacenamiento disponibles en el sistema.

Ahora continuaremos con el disco, pero antes, nos vamos a descar el LM Studio, nos vamos a la página oficial de LM Studio, y nos descargamos nuestra versión, Linux ARM64 y su versión

ahora, mientras se descarga, vamos a seguir configurando nuestro disco. Ahora escribimos sudo mkfs.ext4 /dev/nvme0n1 que este comando formatea y le da la extensión .ext4 a nuestro nvme.

Ahora lo que hacemos es crear un directorio llamado ssd y vamos a montar ahí el disco, así que cuando queremos acceder a el deberémos entrar en el directorio ssd

Ahora ponemos el comando de lsblk -f donde nos mostrará todas las particiones sin modificar nada

Ahora tenemos que entrar en etc/fstab, donde vamos a añadir nuestro disco. Para eso debemos escribir sudo vi /etc/fstab, y escribir lo siguiente: UUID=Tu id del disco /el disco/ formato defaults 0 2 (Le indicamos que de default sea el segundo si el primero no arranca)

Después le decimos al ssd que cambie el propietario del ssd

Ya hemos terminado con esta configuración.
